#  BlogDeCafé

Una página web moderna y responsive diseñada para una comunidad de amantes del café. Este proyecto simula un blog que ofrece consejos, recetas y cursos de formación sobre el mundo del café.


## Descripción del Proyecto
Este proyecto fue desarrollado para practicar maquetación web moderna. El objetivo principal fue crear una interfaz limpia, atractiva y funcional que se adapte correctamente a dispositivos móviles y de escritorio.

La estructura del sitio incluye:
- **Header:** Con navegación y banner principal (Hero section).
- **Blog:** Artículos destacados con imágenes y extractos.
- **Sidebar:** Widget lateral promocionando cursos y talleres.
- **Footer:** Enlaces de navegación rápida.

##  Tecnologías Utilizadas
* **HTML5:** Uso de etiquetas semánticas (`<header>`, `<main>`, `<article>`, `<aside>`, `<nav>`) para una mejor estructura y SEO.
* **CSS3:** Estilos personalizados.
    * **Flexbox:** Para la alineación de elementos en la navegación y componentes internos.
    * **CSS Grid:** Para la estructura general del layout (columnas del blog vs. sidebar).
    * **Variables CSS (Custom Properties):** Para mantener la consistencia en colores y tipografías.
    * **Media Queries:** Diseño "Mobile First" que escala a pantallas de escritorio (768px+).
    * **Normalize.css:** Para asegurar consistencia entre diferentes navegadores.
* **Fuentes:** Integración de Google Fonts (*Open Sans* y *PT Sans*).
